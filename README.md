# CS50 Project 0: Search

A responsive front-end clone of Google Search, Google Image Search, and Google Advanced Search pages, built with HTML and CSS for the CS50 Web Programming course.

## Features

### 1. Google Search (index.html)
- Clean, responsive search interface
- Google logo and search bar
- "Google Search" and "I'm Feeling Lucky" buttons
- Navigation to Image Search and Advanced Search

### 2. Image Search (images.html)
- Dedicated image search interface
- Responsive design that adapts to different screen sizes
- Navigation back to main search

### 3. Advanced Search (advanced.html)
- Advanced search options with multiple input fields
- Formatted to match Google's advanced search layout
- Responsive design with mobile-friendly layout
- Clear visual hierarchy and input organization

## Responsive Design

The website is fully responsive and works on:
- Desktop (screens wider than 768px)
- Tablets (screens between 600px and 768px)
- Mobile devices (screens smaller than 600px)
